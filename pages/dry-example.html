This page demonstrates the client-side JavaScript approach to reducing HTML duplication

What You're Seeing

This page has no hard-coded header or footer HTML. The header above and footer below are dynamically generated by assets/js/components.js.

This means:

  • ‚úÖ Single source of truth for header/footer
  • ‚úÖ Update once, applies to all pages
  • ‚úÖ Dramatically reduced code duplication
  • ‚úÖ Easier maintenance going forward

How It Works

  1. This HTML page has minimal structure (just head + body with content)
  2. The body tag includes data attributes: data-root-page="false"
  3. components.js runs when the page loads
  4. It injects the header and footer HTML dynamically
  5. script.js then initializes all interactive features

‚úÖ New Approach

<body data-root-page="false">
  <!-- Header injected here -->
  
  <main>
    <!-- content -->
  </main>
  
  <!-- Footer injected here -->
  <script src="../assets/js/components.js"></script>
  <script src="../assets/js/script.js"></script>
</body>
